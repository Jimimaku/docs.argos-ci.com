# Getting started

Welcome to the Argos CI documentation !

To learn about Argos philosophy visit [about Argos CI](/docs/about-argos-ci).

### Prerequisites

To use Argos, you need a [GitHub](www.github.com) hosted project with [end-to-end tests](https://kentcdodds.com/blog/static-vs-unit-vs-integration-vs-e2e-tests) and a [CI environment](https://docs.github.com/en/actions/automating-builds-and-tests/about-continuous-integration).

## Install Argos CI

1. Use the integration guide that match your stack or use the [custom integration](/docs/integrations/custom-integration) doc.

2. [Select a plan](https://github.com/marketplace/argos-ci) on Github Marketplace to get your `ARGOS_TOKEN`.

3. Use the CI config guide that matches your stack.

Your are now ready to [add visual testing to your routine](/docs/visual-testing-routine/).
