# Static sites

Fast install guide to static sites.

Use this fast install guide if your stack match of our integrations.

## Gatsby

Use the [`gatsby-plugin-argos`](https://github.com/argos-ci/argos-javascript/tree/master/packages/gatsby-plugin-argos) to fast install Argos in your Gatsby site.

### Fast install

```sh
$ npm install --save-dev gatsby-plugin-argos argos-cli gatsby-plugin-screenshot puppeteer
```

```js
// in your gatsby-config.js
module.exports = {
  plugins: [
    ...(process.env.CI
      ? [
          {
            resolve: 'gatsby-plugin-argos',
            options: {
              branch: process.env.GITHUB_REF_NAME,
              commit: process.env.GITHUB_SHA,
              token: process.env.ARGOS_TOKEN,
            },
          },
        ]
      : []),
  ],
}
```

After each build, every pages will be screenshot and sent to Argos.

Example project : https://github.com/jsfez/argos-gatsby-github-actions
