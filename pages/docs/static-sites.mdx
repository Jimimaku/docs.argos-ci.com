# Static sites

Fast install guide to static sites.

Use this fast install guide if your stack match of our integrations.

## Gatsby

Use the [`gatsby-plugin-argos`](https://github.com/argos-ci/argos-javascript/tree/master/packages/gatsby-plugin-argos) to fast install Argos in your Gatsby site.

### Fast install

<Snippet lang="sh">
  $ npm install --save-dev gatsby-plugin-argos argos-cli
  gatsby-plugin-screenshot puppeteer
</Snippet>

### Expose your `ARGOS_TOKEN` as environment variable

```yml
// Github Actions example: add this command to your .github/workflows/*.yml

env:
  ARGOS_TOKEN: ${{ secrets.ARGOS_TOKEN }}
```

### Add the Argos plugin to Gatsby config

```js
// in your gatsby-config.js
module.exports = {
  plugins: [
    ...(process.env.CI
      ? [
          {
            resolve: "gatsby-plugin-argos",
            options: {
              branch: process.env.GITHUB_REF_NAME,
              commit: process.env.GITHUB_SHA,
              token: process.env.ARGOS_TOKEN,
            },
          },
        ]
      : []),
  ],
};
```

After the build, your CI will take screenshots of every pages and sent them to Argos.

Example project : https://github.com/argos-ci/argos-example-gatsby-github-actions
