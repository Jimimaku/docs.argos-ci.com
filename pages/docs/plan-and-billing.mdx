# Plan and billing

Rely on Github Marketplace

As a GitHub dependent tech product, we chose to rely on GitHub Marketplace to manage users' accounts, payment and billing topics.

We hope that our customers will be happy to rely on safe third party supplier who probably already has your bank details.

Feel free to find bellow some useful links :

- How to [get my Argos token](https://github.com/marketplace/argos-ci)
- How to [subscribe to Argos plan](https://github.com/marketplace/argos-ci)
- How to [update my plan](https://github.com/marketplace/argos-ci)
- How to access [GitHub billing settings](https://github.com/settings/billing)
