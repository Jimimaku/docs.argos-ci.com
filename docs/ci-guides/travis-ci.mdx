# Use Argos with Travis CI

Configure [Travis CI](https://www.travis-ci.com/) to send screenshots to Argos visual testing platform.

## Configuring environment variable

<WhyArgosToken />
<GetArgosToken />

In your Travis repository, go to Settings > Environment Variables. Then, set the `ARGOS_TOKEN` as an environment variable.

<TokenPrivacyWarningMessage />

If you are not comfortable with storing secret, read Gitlab CI documentation about [environment variables](https://docs.travis-ci.com/user/environment-variables/#defining-variables-in-repository-settings).

## Configure Travis CI

```yml title=".travis.yml"
script:
  # ðŸ‘‰ Insert the steps required to take screenshots

  # Upload screenshots to argos-ci.com
  - npx @argos-ci/cli upload ./screenshots
```

Read the [Travis CI repository](https://github.com/argos-ci/argos/tree/main/examples/travis) for a complete configuration example.
