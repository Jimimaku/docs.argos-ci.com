# Use Argos with CodeShip

Configure [CodeShip](https://www.cloudbees.com/products/CodeShip) to send screenshots to Argos.

## Configuring environment variable

<WhyArgosToken />
<GetArgosToken />

In your CodeShip project go to Settings > Environment Variables. Then, set the `ARGOS_TOKEN` as an environment variable.

<TokenPrivacyWarningMessage />

If you are not comfortable with storing secret, read [CodeShip documentation about environment variables](https://docs.cloudbees.com/docs/cloudbees-codeship/latest/pro-builds-and-configuration/environment-variables).

## Configure CodeShip step file

```yml title="codeship-steps.yml"
- name: ci
  type: parallel
  steps:
    - service: project_name
      command: npx @argos-ci/cli upload ./screenshots
```

Read the [CodeShip repository](https://github.com/argos-ci/argos/tree/main/examples/CodeShip) for a complete configuration example.
