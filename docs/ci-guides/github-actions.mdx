# Use Argos with GitHub Actions

Configure [GitHub Actions](https://github.com/features/actions) to send screenshots to Argos.

## Configure GitHub Actions

```yml title=".github/workflows/ci.yml"
name: CI
  build:
    runs-on: ubuntu-latest
      - uses: actions/checkout@v3

      # ðŸ‘‰ Insert the steps required to take screenshots

      - name: Upload screenshots to argos-ci.com
        run: npx @argos-ci/cli upload ./screenshots
```

Read the [GitHub Actions repository](https://github.com/argos-ci/argos/tree/main/examples/github-actions) for a complete configuration example.
