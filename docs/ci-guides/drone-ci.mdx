## Configure Drone CI

```yml title=".drone.yml"
kind: pipeline
type: docker
name: default

steps:
  # ðŸ‘‰ Insert the steps required to take screenshots

  - name: Upload screenshots to argos-ci.com
    commands:
      - npx @argos-ci/cli upload ./screenshots
    secrets: [ARGOS_TOKEN]
```
