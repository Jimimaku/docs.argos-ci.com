# Use Argos with CircleCI

Configure [CircleCI](https://circleci.com/) to send screenshots to Argos visual testing platform.

## 1. Install `@argos-ci/cli`

<InstallCliCommand />

## 2. Set `ARGOS_TOKEN` as CI secret

<WhyArgosToken />

- **Step 1: Get the token value**<br/>
  On Argos, go to the repository you want to set and click on the "settings" tab. Store the ARGOS_TOKEN value.

- **Step 2: Create a secret in your CI**<br/>
  In CircleCI go to **Project settings -> Environment Variables -> Add Environment Variable**. Set the `ARGOS_TOKEN` as an environment variable and past the value you stored in the previous step.

<img
  src={circleCiEnvVariable}
  alt="CircleCI setup environment variable"
  className="rounded"
  style={{ marginBottom: 20 }}
/>

> If you are not comfortable with storing secret, read [CircleCI documentation](https://circleci.com/docs/env-vars/).

## 3. Upload the screenshots

Add the following command to your CI workflow to upload screenshots to Argos:

```sh
npx @argos-ci/cli upload ./screenshots
```

#### Example

```yml title=".circleci/config.yml"
jobs:
  build:
    executor: node/default
    steps:
      - checkout

      # ðŸ‘‰ Insert the steps required to take screenshots

      - run:
          name: Upload screenshots to argos-ci.com
          command: npx @argos-ci/cli upload ./screenshots

workflows:
  ci:
    jobs:
      - build
```
