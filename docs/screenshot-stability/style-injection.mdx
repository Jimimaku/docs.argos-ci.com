# Style injection

Browsers have unstable rendering for well-known use cases like scrollbar display, border-radius, etc. Therefore, it can be convenient to hide these components to ensure the screenshots' consistency.

In our integrations, we inject the following styles before taking a screenshot.

```css
/* Hide carets */
* {
  caret-color: transparent !important;
}

/* Hide scrollbars */
::-webkit-scrollbar {
  display: none !important;
}

/* Generic hide */
[data-visual-test="transparent"] {
  color: transparent !important;
  font-family: monospace !important;
  opacity: 0 !important;
}

[data-visual-test="removed"] {
  display: none !important;
}
```

This snippets hide unstable elements and provide the following attributes you can added to HTML nodes:

- `data-test-hidden` — hide this node from screenshots
- `data-test-no-radius` — remove the border-radius from screenshots

Read the [code injection doc](/code-injection) to know how to inject CSS in a page.
