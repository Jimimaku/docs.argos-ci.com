# Puppeteer Quickstart

Learn how to connect your Puppeteer project to Argos.

## Prerequisites

Before starting, ensure that:

- Puppeteer [is installed](https://pptr.dev/#getting-started)
- A CI/CD run your E2E tests

## 1. Install packages

<InstallDevDep dependency="@argos-ci/cli @argos-ci/puppeteer" />

## 2. Capture screenshots

In your E2E tests, use `argosScreenshot` to capture stable screenshots.

```js
import { argosScreenshot } from "@argos-ci/puppeteer";
const puppeteer = require("puppeteer");

(async () => {
  const browser = await puppeteer.launch();
  const page = await browser.newPage();
  await page.goto("http://example.com");
  await argosScreenshot(page, "example.png");
  await browser.close();
})();
```

**Tip:** Check out our guide to [streamline page screenshot captures](/screenshot-pages-script).
