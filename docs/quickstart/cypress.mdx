# Cypress Quickstart

How to connect your Cypress project to Argos.

## Prerequisites

Before starting, ensure that:

- Cypress [is installed](https://docs.cypress.io/guides/getting-started/installing-cypress) on your project
- A [CI/CD run your E2E tests](https://playwright.dev/docs/ci-intro#on-pushpull_request)

## 2. Capture screenshots

<InstallDevDep dependency="@argos-ci/cli @argos-ci/cypress" />

**II. Copy/past this code at the end of your `cypress/support/index.js` file.**

```js cypress/support/index.js
import "@argos-ci/cypress/support";
```

**III. [optional] Update your `tsconfig.json` to add typings for TypeScript.**

```js
{
  "compilerOptions": {
    "types": ["cypress", "@argos-ci/cypress"]
  }
}
```

**IV. In your E2E tests, use `argosScreenshot` to capture stable screenshots.**

```js title="cypress/e2e/homepage.cy.js"
it("screenshot homepage", async ({ page }) => {
  cy.visit("https://localhost:3000");
  cy.argosScreenshot("homepage");
});
```

<ScreenshotGuidesLink />

**Note:** Consider adding `/cypress/screenshots` to `.gitignore` file, to avoid uploading screenshots to your Git repository.
