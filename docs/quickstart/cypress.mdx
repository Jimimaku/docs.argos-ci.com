# Cypress Quickstart

How to connect your Cypress project to Argos.

## Prerequisites

Ensure that:

- Cypress [is installed](https://docs.cypress.io/guides/getting-started/installing-cypress) in your project.
- A CI/CD setup running your E2E tests

## 1. Install packages

<InstallDevDep dependency="@argos-ci/cli @argos-ci/cypress" />

## 2. Update Cypress config file

Copy/past this code at the end of your `cypress/support/index.js` file.

```js cypress/support/index.js
import "@argos-ci/cypress/support";
```

## 3. Capture screenshots

In your E2E tests, use `argosScreenshot` to capture stable screenshots.

For example:

```js title="cypress/e2e/homepage.cy.js"
it("screenshot homepage", async ({ page }) => {
  cy.visit("https://localhost:3000");
  cy.argosScreenshot("homepage");
});
```

#### TypeScript support

To add typings for TypeScript, update your `tsconfig.json`:

```js
{
  "compilerOptions": {
    "types": ["cypress", "@argos-ci/cypress"]
  }
}
```
