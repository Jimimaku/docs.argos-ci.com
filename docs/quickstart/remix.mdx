# Remix Quickstart

Learn how to connect your Remix project to Argos.

## Prerequisites

To integrate Argos with Remix, ensure Playwright is set up for E2E testing. If using a different E2E tool, see [our specific guides](/getting-started).

Before starting, ensure that:

- You have a [Remix project installed](https://remix.run/docs/en/main/start/tutorial#setup)
- Playwright [is installed](https://playwright.dev/docs/intro#installing-playwright) on your project
- A [CI/CD run your E2E tests](https://playwright.dev/docs/ci-intro#on-pushpull_request)
