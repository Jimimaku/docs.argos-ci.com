# Getting started

Set up Argos for your project in few minutes by following these steps.

<StepList>
  <Step>
    <StepTitle>Create your Argos Account</StepTitle>
    <StepContent>
      Navigate to the <a href="http://app.argos-ci.com/signup">signup page</a>,
      choose a plan, and sign in with your Git provider. <br />
      Authorize Argos app, and you're in!
    </StepContent>
  </Step>
  <Step>
    <StepTitle>Connecting a Repository from your Git Provider</StepTitle>
    <StepContent>
      Either <a href="/github">Github</a> or <a href="gitlab">GitLab</a>.
    </StepContent>
  </Step>
  <Step>
    <StepTitle>Generate Screenshots</StepTitle>
    <StepContent>
      Use your testing tool to{" "}
      <a href="/capture-screenshots">capture screenshots</a> during your E2E
      tests.
    </StepContent>
  </Step>
  <Step>
    <StepTitle>Upload Screenshots</StepTitle>
    <StepContent>
      Add{" "}
      <a href="/upload-screenshots">the Argos command into your CI pipeline</a>{" "}
      to upload the screenshots.
    </StepContent>
  </Step>
  <Step>
    <StepTitle>Review visual changes</StepTitle>
    <StepContent>
      Well done! You are now ready to{" "}
      <a href="/review-changes">review visual changes</a> of your pull requests.
    </StepContent>
  </Step>
</StepList>
