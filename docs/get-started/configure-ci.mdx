# Configure Continuous Integration

To get started with Argos, you'll need to configure your Continuous Integration (CI) environment to use the Argos CLI. This is the easiest and fastest way to upload screenshots to the Argos visual testing platform.

Need help setting up Argos in your CI environment? Check out our [examples repository on GitHub](https://github.com/argos-ci/argos/tree/main/examples) sample configurations.

Don't see your preferred CI listed in our examples? We welcome contributions from the community. Head over to our [contributing guide](https://github.com/argos-ci/argos/blob/main/CONTRIBUTING.md) to learn how you can add an example for your CI.

## Get Started with Visual Testing

Congratulations, you have successfully set up Argos in your CI environment. You are now ready to begin your visual testing routine. Head over to our [visual testing](/visual-testing) section to learn more about how to use Argos to detect and fix visual regressions in your web applications.
