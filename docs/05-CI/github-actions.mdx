# Github Actions

Argos automatically authenticates your repository in GitHub Actions, you don't need to specify a token.

Add this command to your CI to safely send your screenshots to Argos.

```
npx @argos-ci/cli upload ./screenshots
```

> Replace `.\screenshots` with the folder path that containing the screenshots.
